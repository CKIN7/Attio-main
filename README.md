# Attio

Sleek and modern CRM website built with Next14, Shadcn, and Tailwind.

# ☁️ Deployed link

https://attio-three.vercel.app/

# ⛏️ Installation

Install the npm packages & run

```bash
npm install && npm run dev
```

## 🔓 Stack

-   Next14
-   Typescript
-   Shadcn
-   Tailwind

## 🔓 Benefits

-   Mobile Responsiveness
-   Learn all the stack in one video
